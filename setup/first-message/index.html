第一次成功不是“看懂全部”，而是完成一次真实可验证的交互

只要你能在目标入口和 OpenClaw 正常互动，并确认背后的链路是通的，这次 setup 就算过了第一关。

第一次成功应包含什么

  • 你知道消息从哪里发出。
  • 你知道系统在哪里响应。
  • 你能判断这不是假象，而是真正完成了一次可用交互。

建议验证项

1

发出一条简单请求

例如让系统回答一个基础问题。

2

确认响应回来

并确认不是卡在 gateway / UI / token 层。

3

观察最小运行闭环

知道这条消息经历了入口、session、模型、工具（如有）和返回。
