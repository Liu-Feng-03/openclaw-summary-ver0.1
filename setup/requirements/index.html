至少要确认什么

  • 操作系统与基础运行环境是否受支持。
  • Node / 包管理器 / 终端权限是否正常。
  • 是否能访问后续需要的模型/API/网络资源。
  • 如果要用 Dashboard、消息平台或外部集成，相关前提是否满足。

成功信号

  • 运行环境版本明确。
  • 终端可正常执行基础命令。
  • 没有明显的权限或网络阻断前提。
