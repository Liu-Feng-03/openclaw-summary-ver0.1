Setup

先确认环境，再开始安装

很多“安装失败”其实不是安装命令错了，而是系统、运行时、权限或依赖前提没满足。

继续安装
Wave 1当前优先覆盖理解 → 配置 → 首次成功闭环。
多页面每个重要主题都有独立 URL，便于分享与扩展。
分层阅读所有页面都遵循“短答案 → 解释 → 下一步”的节奏。
这一步的价值不是复杂，而是避免你后面浪费时间。

你至少要确认什么

  • 操作系统与基本运行环境是否受支持。
  • Node / 包管理器 / 终端权限是否正常。
  • 你是否能访问后续需要的模型/API/网络资源。
  • 如果要用 Dashboard、消息平台或外部集成，相关入口条件是否满足。

成功信号

  • 运行环境版本明确。
  • 终端可正常执行基础命令。
  • 没有明显的权限或网络阻断前提。
