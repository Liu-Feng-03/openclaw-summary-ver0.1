Core Concepts

Dashboard 是观察和操作 OpenClaw 的控制面

它不是系统本体，但能让你更方便地查看状态、切换界面、验证连接与操作会话。

中文学习站 可分享 URL 从入门到进阶

48已覆盖 roadmap 中首页、入门、概念、配置、场景、进阶、FAQ 全部模块。
多页面每个重要主题都有独立 URL，方便分享与后续扩写。
分层阅读页面都遵循“短答案 → 解释 → 下一步”的学习节奏。
理解 dashboard 的定位，有助于区分“界面层问题”和“系统层问题”。

它的作用

  • 观察状态。
  • 进入控制界面。
  • 辅助验证 gateway、token、连接情况。
