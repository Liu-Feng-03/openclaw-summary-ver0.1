Getting Started

OpenClaw 是一套可接入真实工具与渠道的 AI 助理系统

它不是单纯在网页里和你聊天，而是让代理、会话、工具、技能、记忆和外部入口一起协作。

先把它理解成“有结构、有边界、可操作”的助理平台，会比把它当成普通聊天机器人更接近真实。

一句话定义

  • OpenClaw 是一个让 AI 代理在真实消息渠道、工具系统与文件记忆上运行的助理框架。

它解决什么问题

  • 让助理不只会回答，还能读取文件、调用工具、连接 Telegram / Discord / GitHub 等入口。
  • 把长期协作需要的“人格、规则、记忆、执行能力”放到可管理的结构里。
  • 让不同任务可以用不同 session、不同 skill、不同权限和不同工作流来处理。

它和普通聊天 UI 的区别

普通聊天 UI

重点是对话本身，能力边界常常不透明。

OpenClaw

重点是“对话 + 系统结构 + 工具执行 + 渠道接入 + 持续协作”。
